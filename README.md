
# Applications

### 1. Kasi Library: app

This the application that will allow you to add and remove books.

### 2. Factorial: app
This is an application that sping up multiple threads to calculate the factorial of 10

### 3. Stream: app
This is the application that will produce number in the queue and allow the consumers to eat them.


### 6. Kasi Weather Weather Report: app
This is the application that will call the the **open-meteo** to get the weather of the City.


## 5. Kasi Shop : app
This is the Rest API application that will add products to your MySQL Database. There is unit test included

Command to apply the database migration :
```
Update-Database
```
